# Code of Conduct

## Our Goal

这个项目希望保持一个直接、友好、可协作的讨论环境。

欢迎提交 bug、需求、PR 和使用反馈，但请尽量围绕问题本身沟通，避免把讨论变成人身冲突。

## Expected Behavior

- 尊重不同经验水平的贡献者
- 聚焦代码、行为和事实
- 接受技术分歧，但避免情绪化攻击
- 对 bug 复现、风险和证据保持认真
- 对用户反馈保持基本耐心

## Unacceptable Behavior

- 人身攻击、羞辱、挑衅
- 歧视性言论
- 恶意刷屏、骚扰、引战
- 明知会造成破坏仍反复提交无关或有害内容
- 泄露他人隐私、凭据或本地敏感信息

## Enforcement

仓库维护者可以根据情况删除评论、关闭讨论、拒绝 PR，或限制参与权限。

## Scope

本行为准则适用于本仓库的 Issues、Pull Requests、Discussions 和其他公开协作场景。
