# Contributing

感谢你愿意帮助改进这个项目。

这个仓库是 [GenericAgent](https://github.com/lsdefine/GenericAgent) 的桌面启动器，目标是让上游项目更容易下载、配置、启动和日常使用。为了减少回归和沟通成本，提交前请先看完下面这些约定。

## 提交前建议

1. 先确认问题是否已经存在于 `Issues` 或 `Discussions`。
2. 变更前尽量先描述问题、复现方式和你的修复思路。
3. 如果改动会影响界面交互，尽量附截图或录屏。
4. 如果改动会影响打包行为，请说明是否已经验证 `exe`。

## 开发环境

```bash
pip install -r requirements.txt
python launcher.py
```

重新打包：

```bash
build.bat
```

## 变更原则

- 优先修复真实问题，不做无关重构。
- 尽量保持现有界面风格，不随意重做视觉设计。
- 用户非常敏感于交互回归，尤其是：
  - 侧边栏收起/展开
  - 右上角工具面板定位
  - 会话列表和右键菜单
  - 聊天渲染折叠
  - 打包后 exe 与源码不一致
- 改动涉及上游 GenericAgent 兼容性时，请说明你参考了哪些上游配置或行为。

## 提交内容建议

- Bug 修复请写清楚：
  - 问题现象
  - 复现步骤
  - 修复方式
  - 风险点
- 新功能请写清楚：
  - 使用场景
  - 是否影响现有配置
  - 是否需要新增依赖

## Pull Request 建议

- 保持 PR 尽量聚焦，一个 PR 解决一类问题。
- 标题请直接描述结果，而不是只写 “fix bug”。
- 如果你没有实际验证，请明确说明“未验证”。

## 沟通方式

- 功能建议、产品方向、交互讨论更适合走 `Discussions`
- 可复现的 bug、明确的改进事项更适合走 `Issues`

## 许可证

向本仓库提交代码，默认表示你同意你的贡献以本仓库当前的 MIT License 发布。
